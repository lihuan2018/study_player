# 美剧学习播放器

一个功能强大的美剧学习播放器，支持本地视频和字幕文件导入，智能字幕交互，以及灵活的播放控制，帮助您更有效地学习英语。

## 功能特点

### 核心功能

- **本地视频文件导入**：支持 MP4、MOV、AVI、MKV、WebM 等格式
- **本地字幕文件导入**：支持 SRT、VTT、ASS、SSA 格式
- **智能字幕交互**：点击字幕跳转到对应时间点，当前字幕自动高亮并居中显示
- **灵活播放控制**：整段循环播放，A-B 片段循环重复
- **个人中心**：版本信息、联系我们、清除缓存等功能
- **跨平台兼容**：支持 H5 环境和 App 环境
- **响应式设计**：适配不同屏幕尺寸

### 技术特点

- **流畅的用户体验**：清晰的界面布局，直观的操作方式
- **强大的错误处理**：优雅处理各种异常情况
- **智能字幕解析**：自动识别字幕格式，支持多种编码
- **高性能**：优化的字幕滚动和视频控制

### 使用说明

1. **底部导航栏**：
   - **首页**：视频播放器功能，用于播放视频和查看字幕
   - **我的**：个人中心功能，包含版本信息、联系我们、清除缓存等
2. **加载视频文件**：
   - 点击视频文件上传区域
   - 在 H5 环境下，会打开文件选择器
   - 在 App 环境下，会调用系统文件选择器
   - 支持拖拽文件到上传区域（仅 H5 环境）
3. **加载字幕文件**：
   - 点击字幕文件上传区域
   - 选择 SRT、VTT、ASS 或 SSA 格式的字幕文件
   - 系统会自动解析字幕内容并显示在字幕列表中
4. **播放控制**：
   - 使用视频播放器自带的控制条进行播放、暂停、音量调节等操作
   - 点击 "循环" 按钮开启/关闭循环模式
   - 查看当前播放时间和视频总时长
5. **A-B 循环设置**：
   - 播放视频到需要的起点，点击 "起点 A" 按钮设置循环开始点
   - 播放视频到需要的终点，点击 "终点 B" 按钮设置循环结束点
   - 开启循环模式后，视频会在 A-B 区间内循环播放
   - 点击 "🗑️" 按钮清除循环点
6. **字幕交互**：
   - 点击字幕列表中的任意字幕，视频会跳转到对应时间点并开始播放
   - 播放过程中，当前播放的字幕会自动高亮显示并居中滚动
7. **个人中心功能**：
   - **版本信息**：查看当前应用版本号（与manifest.json保持一致）
   - **联系我们**：查看联系方式
   - **清除缓存**：清除应用缓存（仅 App 环境支持）
   - **推荐内容**：展示英语学习资源推荐

## 技术实现

### 视频播放

- 使用 `uniapp-video-player` 插件实现跨平台视频播放
- 在 H5 环境下直接操作原生 video 元素，支持更丰富的控制
- 在 App 环境下使用插件提供的 API 控制视频
- 实现了视频加载状态管理和错误处理

### 字幕解析

- 支持解析 SRT、VTT、ASS、SSA 格式的字幕文件
- 智能识别字幕格式，自动处理不同的时间戳格式
- 自动跳过字幕文件开头的多余内容，从第一个有效字幕块开始解析
- 自动过滤字幕中的广告内容和网址链接（如论坛链接、微博链接、公众号信息等）
- 自动过滤纯装饰符号（如 ■ 、 ★ 、 ● 等）单独成行的内容
- 清理字幕文本中的乱码和特殊标签
- 构建结构化的字幕数据，支持快速查询和跳转

### 循环播放

- 实现整段循环：视频播放结束后自动重新开始
- 实现 A-B 片段循环：在指定的时间区间内循环播放
- 在 H5 和 App 环境下分别使用不同的实现方式
- 提供直观的循环状态显示

### 字幕交互

- 实现字幕点击跳转功能，支持精确到秒的时间定位
- 自动高亮当前播放的字幕
- 使用 scroll-into-view 实现字幕自动居中滚动
- 采用节流优化，提升滚动性能

### 文件选择

- 在 H5 环境下使用 input\[type=file] 实现文件选择
- 在 Android 环境下使用 Native.js 调用原生文件选择器
- 在其他 App 环境下使用 uni.chooseMessageFile API
- 实现了权限请求和错误处理

## 常见问题

### 1. 视频加载失败

- 检查视频文件格式是否支持
- 检查视频文件是否损坏
- 尝试使用 MP4 格式的视频文件
- 检查浏览器或设备是否支持该视频编码

### 2. 字幕解析失败

- 检查字幕文件格式是否支持
- 检查字幕文件编码是否为 UTF-8
- 尝试使用 SRT 格式的字幕文件
- 检查字幕文件内容是否规范

### 3. 循环播放不起作用

- 确保已开启循环模式
- 确保已设置有效的循环点（A-B 循环）
- 检查视频是否正常播放
- 尝试重新设置循环点

### 4. 字幕点击跳转不起作用

- 确保已加载视频文件
- 确保已加载字幕文件
- 检查字幕文件是否解析成功
- 检查视频是否已准备就绪

### 5. 文件选择失败

- 检查设备是否授予文件访问权限
- 确保文件路径正确
- 尝试使用不同的文件选择方式
